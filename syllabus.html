Contenu et objectifs pédagogiques

Ce cours est une initiation à la programmation web dynamique en utilisant les technologies HTML, CSS et Javascript. Nous verrons comment utiliser ces outils afin de créer une application web hébergée sur GitHub au service des humanités numériques à travers des applications simples de traitement automatique des langues.

Les objectifs pédagogiques du cours sont les suivants :

  • apprendre à créer une page web en HTML ;
  • la déployer sur le web à l’aide des GitHub pages ;
  • changer le style d’une page web avec CSS ;
  • utiliser Javascript pour créer du contenu dynamique sur une page web ;
  • appliquer ces connaissances aux traitements automatiques des textes.
